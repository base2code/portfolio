Minecraft Cloud System

A custom Minecraft cloud system for the LyroTopia.net network, using Docker to run the underlying spigot and proxy services.

Project Overview

Initially we at LyroTopia.net decided to use CloudNet. However, we encountered certain challenges that led us to develop our own unique cloud system, perfectly tailored to our needs. During the development phase, we considered implementing containers to isolate different servers and manage network access more efficiently.
